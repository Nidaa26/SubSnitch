{# Reusable subscription form fields shared by the add and edit pages. Expects: * `form_data` - a mapping (request.form on error, a Subscription on edit, or {} on a fresh add) used to pre-fill the fields. * `categories` - list of selectable categories. * `billing_cycles` - list of billing cycle options. A small macro reads a value from `form_data` whether it is a dict-like form object or a model instance, so the same partial works everywhere. #} {% macro field_value(data, key) -%} {%- if data is mapping -%} {{ data.get(key, '') if data.get(key) is not none else '' }} {%- else -%} {{ data[key] if data[key] is not none else '' }} {%- endif -%} {%- endmacro %} {% set v_name = field_value(form_data, 'name') | trim %} {% set v_category = field_value(form_data, 'category') | trim %} {% set v_billing = field_value(form_data, 'billing_cycle') | trim %} {% set v_price = field_value(form_data, 'price') | trim %} {% set v_uses = field_value(form_data, 'uses') | trim %} {% set v_notes = field_value(form_data, 'notes') | trim %} {# Dates need ISO formatting when coming from a model instance. #} {% if form_data is mapping %} {% set v_renewal = field_value(form_data, 'renewal_date') | trim %} {% set v_last_used = field_value(form_data, 'last_used') | trim %} {% else %} {% set v_renewal = form_data.renewal_date.strftime('%Y-%m-%d') if form_data.renewal_date else '' %} {% set v_last_used = form_data.last_used.strftime('%Y-%m-%d') if form_data.last_used else '' %} {% endif %}
